# Radio Signal Analysis Project

## Overview

This project simulates and processes radio astronomy signals in order to detect and analyze spectral lines (hydrogen line at 1420 MHz with bandwidth of < 5 MHz).

The workflow includes:

* Signal simulation
* Noise modeling
* Signal smoothing
* Calibration (on/off)
* Spectral line detection
* Upsampling
* Curve fitting
* Error estimation

## Structure

* `notebooks/`: Main analysis notebook
* `src/`: Modular implementation
* `results/`: Figures and evaluation metrics

## Goals

The goal is to demonstrate:

* Signal processing understanding
* Clean data pipeline design
* Visualization and interpretation skills

## Technologies

* Python
* NumPy
* SciPy
* Matplotlib
